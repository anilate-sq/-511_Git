# Основные команды
## restore
```bash
git restore <файл> # Откат изменений в рабочей директории
git restore --staged <файл> # Откат изменений исключительно из индекса
git restore --staged --worktree <file> # Одновременный откат из индекса и из рабочей директории
```
## reset

```bash
git reset <коммит> # --mixed режим, откат к указанному коммиту с сохранением изменений в рабочей директории
git reset --soft<коммит> # --soft режим, откат к указанному коммит с сохранением изменений в индексе
git reset --hard<коммит> # --hard режим, откат к указанному коммиту без сохранения изменений
```
