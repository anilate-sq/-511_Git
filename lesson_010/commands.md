# Основные команды для работы

```bash
git clone <ссылка> # Клонирует удаленный репозиторий в папку из которой была выполнена команда
git push <ссылка>/origin <ветка> # Отправка изменений в удаленный репозиторий
git pull <ссылка>/origin <ветка> # Получение всех изменений с удаленного сервера на локальную папку
git pull/push -u <ссылка>/origin <ветка> # Git запоминает ссылку и ветку, после чего в последующих операциях по отправке или получению их ненужно указывать
```
