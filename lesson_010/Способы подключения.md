# Способы подключения

1. Когда мы создаем новый проект;

```bash
echo "# Test" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/anilate-sq/Test.git
git push -u origin main
```

2. Когда мы хотим подключиться к существующему проекту

```bash
git remote add origin https://github.com/anilate-sq/Test.git
git branch -M main
git push -u origin main
```
