# Список ключевых комманд

### Слияние веток
```bash
git switch <ветка> # Переключение на новую ветку
git merge <ветка> # Слить изменения
```

### Проверка статуса и состояния
```bash
git status # Показать конфликты
git diff # Показать различия
git log --graph # Просмотреть граф коммитов
```

### При конфликте
```bash
git merge --abort # Отмена слияния(лучше использовать его для отмены слияния)
git reset --hard HEAD~1 # Отмена слияния(более жесткое, удалятся вообще все изменения)
git mergetool # Открытие визуального инструмента
git add <файл> # Добавление файла в интекс
git commit # Завершения слияния
```
