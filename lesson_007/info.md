# Конфликты слияния

**Конфликт слияния** - конфликт, возникаемый при провидении слияния веток, когда git не может автоматически их слить.

**Причины возникновения конфликта:**
1. Одна и таже строка изменена в двух ветках 
2. Файл изменен в одной ветке и удален в другой ветке
3. Два файла переименованы в одно имя

## Пошаговое разрешение конфликта
1. Получить сообщение о конфликте
```
$ git merge <вливаемая ветка>
Auto-merging <файл, в котором возник конфликт>
CONFLICT ...
```
2. Просмотр статуса
3. Ручное редактирование
4. Добавление исправленного файла в индекс
5. Сделать коммит, тем самым завершив слияние
