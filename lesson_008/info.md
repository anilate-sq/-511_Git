# Rebase(перезапись истории)
**Rebase** - это операция перемещения ветки в другую базовую точку.

### Как работает Rebase:
1. Git находит **общего предка** двух веток
2. Сохраняте изменения из ветки наборами
3. Времено удаляет комиты
4. Перещает ветку на новую базовую ветку
5. Применяем сохраненные набора по одному
