# Практическое задание: Конфликты слияний и Rebase

### Задание 1.
1. Создать любой текстовый файл
2. Воссоздать в нем конфликт слияний
3. Разрешить конфликт слияний
4. Отправить результат набором скриншотов

### Задание 2.
1. Создать 3 файла в ветке master
2. Создать 3 файла в ветке dev
3. Произвести базовый Rebase(перезапись)
4. Результат отправить архивом или скриншотами

### Задание 3.
1. Создать любой текстовый файл
2. Воссоздать в нем конфликт перезаписи
3. Разрешить конфликт перезаписи
4. Отправить результат набором скриншотов
