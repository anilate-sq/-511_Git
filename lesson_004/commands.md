## Ключевые команды
- **`git add <file>`** - добавление в индекс
- **`git diff`** - просмотр изменений
- **`git restore <file>`** - отмена измений

## Команды для работы с просмотром изменений
- **`git log`** - просмотр истории
- **`git log --oneline`** - Полная история в кратком отображании
- **`git log --graph`** - Отображение истории в виде графа
- **`git log --stat`** - Краткое описание изменений
- **`git log -n <число>`** - Количество отображаемых коммитов
- **`git log <file>`** - История изменений определенного файла
- **`git log --pretty=format:"%h - %an, %ar : %s"`**
- **`git diff`** - просмотр изменений
- **`git diff --staged`** - просмотр изменения в индексе
- **`git diff HEAD~1 HEAD`** - просмотр изменений между ближайшими коммитами
- **`git diff <branch1> <branch2>`** - просмотр измененй между ветками
- **`git diff <file>`** - просмотр изменений в рамках определенного файла
