# Команды по работе с ветками

### Создание веток
- **`git branch <название>`**  - создание новой ветки
- **`git branch -b <название>`** создание новой ветки и переключение на неё
- **`git checkout -b <название>`** - создание новой ветки(старый синтаксис)  
- **`git switch -c <название>`** - создание и переключение на новую ветку

### Переключение веток
- **`git switch <название>`** - переключиться на новую ветку
- **`git checkout <название>`** - переключение на указанную ветку(старый синтаксис) 
-- **`git switch -`** - переключение на предыдущую ветку

### Просмотр веток
- **`git branch`** - отображение списка локальных веток
- **`git branch -a`** - отображение списка всех веток
- **`git branch -v`** - отображение списка веток с хэшом и сообщением коммита
- **`git branch -vv`** - отбражение всех веток с remote-треками
- **`git show-ref --heads`** - просмотр ref-ссылок

### Удаление веток
- **`git branch -d <название>`** - безопасное удаление
- **`git branch -D <название>`** - принудительное удаление
